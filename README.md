# mysql-website-list

---

## Technologies Used
- Node.js
- MySQL
- mysql2 package
- JavaScript

---

## Prerequisites
- Node.js (v14 or later recommended)
- MySQL (Ensure MySQL server is running)
- npm (comes with Node.js)

---

## Installation
1. 
```bash
git clone https://github.com/SamuelhaIIman/mysql-website-list.git
```
2. 
```bash
npm install ejs express mysql2
```
